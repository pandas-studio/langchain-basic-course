# ABC 주식회사 — IT 사용자 가이드

본 문서는 ABC 주식회사 임직원의 IT 환경 사용 안내서입니다.
긴급 IT 지원이 필요하시면 IT 헬프데스크(내선 1700)로 연락주세요.

---

## 1. 계정 및 인증

### 1.1 사번 기반 SSO 계정
- 모든 사내 시스템(메일·HR·재무·코드 저장소)은 **SSO(통합 로그인)** 으로 접근
- 사번이 곧 SSO ID (예: `E12345`)
- 비밀번호는 **90일마다** 자동 만료 → 변경 알림 후 7일 내 갱신

### 1.2 비밀번호 정책
- 최소 12자 이상
- 영문 대소문자 + 숫자 + 특수문자 조합 필수
- 직전 5개 비밀번호 재사용 금지
- 사번·이름·생일은 비밀번호로 사용 금지

### 1.3 OTP (이중 인증)
- 모든 SSO 로그인에 **OTP 필수**
- 권장 앱: Google Authenticator, Microsoft Authenticator
- 분실 시 IT 헬프데스크에 신원 확인 후 재등록 (사원증 + 본인 확인)

## 2. VPN

### 2.1 권장 클라이언트
- **FortiClient** (Windows/Mac/Linux 공식 지원)
- 다운로드: 사내 IT 포털 → 소프트웨어 → FortiClient

### 2.2 VPN 접속 방법
1. FortiClient 실행
2. 게이트웨이 주소: `vpn.abc-example.com`
3. 사번 + SSO 비밀번호 입력
4. OTP 입력
5. 연결 확인 후 사내망 자원 접근

### 2.3 VPN 사용 권장 사항
- **재택근무 중** 사내 시스템 접속 시 VPN 필수
- 카페·공항 등 공용 Wi-Fi 사용 시 VPN 강력 권장
- 사용 후 반드시 종료 (트래픽 한도 절약)

## 3. 자주 발생하는 문제 및 해결

### 3.1 VPN 접속 불가
| 증상 | 해결 방법 |
|------|----------|
| 인증 실패 | 비밀번호 만료 확인 → SSO 포털에서 갱신 |
| 연결 시도 중 멈춤 | 다른 네트워크(예: 모바일 핫스팟) 시도 |
| OTP 거부 | 휴대폰 시간 동기화 확인 (자동 시간으로 설정) |
| 모든 방법 실패 | IT 헬프데스크 (내선 1700 / ithelp@abc-example.com) |

### 3.2 노트북 부팅 안됨
1. 전원 어댑터 연결 확인
2. 30초 이상 전원 버튼 눌러 강제 종료 후 재시도
3. 외장 USB·HDMI 모두 분리 후 재시도
4. IT 헬프데스크에 AS 신청 (티켓 등록)

### 3.3 메일 첨부파일 차단
- 25MB 초과 파일은 **사내 파일공유 시스템** 사용 (`files.abc-example.com`)
- `.exe`, `.bat`, `.zip` 안의 실행파일은 보안 정책으로 자동 차단
- 외부 발신 시 첨부파일 자동 패스워드화 (수신자에게 별도 안내)

## 4. 노트북·PC 관리

### 4.1 신규 지급 노트북
- 입사 첫날 IT팀에서 사번과 함께 노트북 지급
- 기본 지급: MacBook Pro 14" / Dell XPS 13 / ThinkPad X1 (직무별 선택)
- 4년 사용 후 교체 (단, 고장 시 즉시 교체)

### 4.2 AS 신청
1. IT 헬프데스크 포털(`ithelp.abc-example.com`) 접속
2. 티켓 등록: 증상 / 발생 시점 / 영향 범위
3. SLA: 일반 24시간, 긴급 4시간 내 응답

### 4.3 노트북 반납
- 퇴사일 기준 D-3까지 IT팀에 반납
- 데이터 마이그레이션은 본인이 사전 진행
- 반납 시 비밀번호 해제, 디스크 검사 후 인수확인서 작성

### 4.4 분실·도난
- 즉시 IT 헬프데스크에 신고 (24시간 가능)
- 원격 잠금 + 데이터 wipe 진행
- 경찰 신고 사본 IT팀에 제출 (사후 처리용)

## 5. 보안 정책

### 5.1 외부 USB 사용
- USB 메모리 사용 **원칙적 금지**
- 업무상 필요한 경우 IT 보안팀 사전 승인 필수
- 승인된 USB만 사용 가능 (암호화 USB 한정)

### 5.2 클라우드 서비스
- 회사 데이터 외부 클라우드(개인 Gmail/Dropbox/Google Drive) 업로드 금지
- 승인된 클라우드: Microsoft 365, AWS S3 (회사 계정), Confluence
- ChatGPT/Claude 등 LLM 서비스에 **회사 기밀 입력 금지**

### 5.3 화면 잠금
- 자리 비울 때는 반드시 `Cmd+Ctrl+Q` (Mac) / `Win+L` (Windows) 화면 잠금
- 5분 무활동 시 자동 잠금 (정책 강제)

### 5.4 보안 사고 신고
- 의심스러운 메일·링크 클릭 시 즉시 IT 보안팀 보고
- 절대 동료에게 비밀번호 공유 금지
