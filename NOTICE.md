## 1. ✅ 자유롭게 허용되는 사용

다음 용도는 **별도 동의 없이 자유롭게** 사용할 수 있습니다.

- **개인 학습**: 노트북을 본인 PC에 clone 하여 직접 실행·수정·학습
- **비공개 메모**: 본인 블로그·노션·옵시디언 등에 학습 노트로 정리
- **출처 표시 인용**: 짧은 코드/문장 인용 시 다음 형식으로 출처 명시
  ```
  출처: 판다스 스튜디오 — langchain-basic-course
  https://github.com/pandas-studio/langchain-basic-course
  ```
